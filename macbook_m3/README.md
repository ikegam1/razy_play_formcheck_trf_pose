# Mac Pose TimeShift & Recorder

Macのインカメラ（または外部Webカメラ）を利用して、リアルタイムで骨格推定（Pose Estimation）を行いながら、**「遅延再生（タイムシフト）」**と**「バックグラウンド録画（等倍速保存）」**を完全に同時実行できるスポーツ・フォームチェック用のアプリケーションです。

屋外や通信環境のない場所でも完全にスタンドアロンで動作するため、陸上競技（ランニング・マラソン）のフォーム確認、各種球技、トレーニング、ガジェットの動作確認などに最適です。

---

## 🌟 主な機能と特徴

1. **リアルタイム骨格推定（MediaPipe Tasks API）**
   - Google MediaPipe の最新の Landmarker API を使用し、全身の主要な関節ポイントと骨格リンクをリアルタイムに描画します。
   - Apple Silicon (M1/M2/M3) のハードウェア性能を活かし、高解像度（1280x720）でも低遅延で処理します。

2. **オンデマンド・タイムシフト再生（遅延再生）**
   - 画面上のボタン（`< 10s`, `< 20s`, `< 30s`）をクリックするだけで、いつでもその秒数だけ過去に遡って自分の動きをリプレイできます。
   - リプレイが終わると、自動的に最新のライブ映像表示へとシームレスに戻ります。

3. **裏での最新ライブ映像の継続録画（完全同時処理）**
   - 画面上で過去の映像を再生（PLAYBACK）して確認している最中であっても、バックグラウンドでは**「今カメラの前に映っている最新の動き」**を骨格付きで途切れなく録画し続けます。

4. **実測FPS自動追従による等倍保存（Mac環境最適化）**
   - Macのインカメラ特有の可変フレームレートやAI処理の負荷を考慮し、処理速度（実際の動作FPS）をミリ秒単位でリアルタイムに自動計測します。
   - `REC` 開始時の実測FPSに基づいて動画を生成するため、録画されたMP4ファイルが早送り（倍速）になってしまう問題を解消し、現実と同じ等倍スピードで保存されます。

5. **完全オフライン対応**
   - 初回起動時に骨格推定用のAIモデルファイル（`pose_landmarker_full.task`）をローカルに自動ダウンロードします。
   - 2回目以降の起動や、AIの推論処理、動画の書き込みにはインターネット接続を一切要求しないため、屋外の練習場などでも安心して使用できます。

---

## 🛠️ 必要要件 (Requirements)

- **OS**: macOS (Apple Silicon M1/M2/M3 シリーズ推奨、Intel Macも可)
- **Python**: 3.9 〜 3.11 (MediaPipeの対応バージョン)
- **カメラ**: 内蔵FaceTimeカメラ、または外部USB Webカメラ
